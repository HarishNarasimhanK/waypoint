# Waypoint — Vision & Problem Statement

## The Problem

Off-campus job applications are getting harder day by day. People find it difficult to:

1. **Discover jobs early** — By the time you see a posting on LinkedIn, hundreds have already applied
2. **Evaluate quickly** — Switching between tabs, reading JDs, comparing with your resume is tedious
3. **Manage applications** — Tracking what you applied to, where you stand, follow-up dates — it all gets lost
4. **Optimize per job** — One resume doesn't fit all, but tailoring for every application is time-consuming

## The Solution

Waypoint is a web application that provides:

1. **Job Radar** — Intelligent scraper that surfaces fresh jobs as soon as they appear on portals
2. **Full Job View** — Complete job details (description, qualifications, everything) in one place
3. **Resume Optimizer** — Upload your resume, compare against a specific JD, get actionable tweaks
4. **Application Tracker** — Simple status management so nothing falls through the cracks

## Core Principles

- **Free-first**: Core features work with open-source models. Paid APIs are pluggable for premium experience.
- **Incremental**: Ship small, working increments. Each feature is useful standalone.
- **Extensible**: Plugin architecture — new job boards, AI providers, notification channels can be added without touching core code.
- **Simple**: A fresher with no technical background should be able to use it immediately.

## Tagline

> "Every goal needs a waypoint."

## Target User

Fresh graduates and early-career professionals actively applying to off-campus tech jobs who are overwhelmed by the daily grind of job hunting.
